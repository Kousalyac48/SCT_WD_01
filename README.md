# 🚀 Sadhana Club 2026 | Landing Page

A high-performance, responsive portfolio website built for **Sadhana Club 2026**. This project showcases modern web development techniques including Glassmorphism, smooth animations, and interactive UI components.

## ✨ Features
- **Modern UI/UX:** Clean design using the Plus Jakarta Sans typeface and a professional 135° gradient aesthetic.
- **Glassmorphism:** Navigation and components feature frosted-glass effects using `backdrop-filter`.
- **Typewriter Effect:** An interactive hero section that cycles through core values: Innovation, Excellence, and The Future.
- **Infinite Skills Scroller:** A seamless, auto-scrolling marquee showcasing technical expertise.
- **Interactive Project Cards:** Hover-reveal overlays with detailed project descriptions and action buttons.
- **Fully Responsive:** Optimized for all screen sizes from mobile devices to desktop monitors.

## 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Flexbox, Grid, Keyframe Animations)
- **Interactivity:** Vanilla JavaScript
- **Icons:** Font Awesome 6.0
- **Fonts:** Google Fonts (Plus Jakarta Sans)
